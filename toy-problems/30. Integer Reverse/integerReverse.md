# 30. Integer Reverse

## About the Problem

Given a positive integer, return its digits reversed.

- DO NOT CONVERT THE INPUT INTO A STRING OR ARRAY.
- Only use integers and math in your solution.

### Examples

| Example| Input | Output |
| --- | --- | --- |
| 1 | `1234` | `4321` |
