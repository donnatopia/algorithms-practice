# 13. All Anagrams

## About the Problem

Given a single input string, write a function that produces all possible anagrams of a string and outputs them as an array.

## Extra Credit:

No repeated strings

### Examples

| Example| Input | Output |
| --- | --- | --- |
| 1 | `abc` | `[ 'abc', 'acb', 'bac', 'bca', 'cab', 'cba' ]` |
| 2 | `a` | `['a']` |
