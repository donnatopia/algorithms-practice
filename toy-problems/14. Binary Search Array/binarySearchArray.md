# 14. Binary Search Array

## About the Problem

| --- | --- | --- |
| 1 | `binarySearch([1, 2, 3, 4, 5], 4)` | `3` |
| 2 | `binarySearch([1, 2, 3, 4, 5], 8)` | `null` |
